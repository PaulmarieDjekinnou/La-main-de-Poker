# La-main-de-Poker

Le projet que nous allons livrer correspond quasiment à un jeu de poker comme on nous l'avait demandé.
Il contient la saisie de 5 cartes par chaque joueur, l'identification des différentes types de mains, 
l'identification de toutes les combianaisons du jeu. Ce qui démontre que nous avons réussi
à implenter toutes les différentes comparaisons possible, ce qui vous permettra d'avoir toujours un gagant ( un résultat ).
Nous avons mis assez de tests afin de valider la solidité de notre projet.
